This sample mobile app demonstrates usage of readLocal, autoSave properties pertain to JSDO DataSource (or JSDO Dialect) which are used in integration with Kendo DataSource. This mobile app is built using Telerik Platform ™ and uses a Progress Data Service.

The built-in session management and data access to a Progress Data Service is a jumpstart to building your mobile app(s) for a Progress OpenEdge or Progress Rollbase server.
